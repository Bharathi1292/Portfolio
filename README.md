## Features

- **About Me:** Introduction and professional background.  
- **Technical Skills:** Programming languages, tools, cloud, databases, and visualization.  
- **Projects:** Highlight of one major project with GitHub link.  
- **Certifications & Achievements:** Verified certificates with clickable links.  
- **Hobbies & Interests:** Personal activities and interests.  
- **Testimonials:** Mentor feedback from internship experience.  
- **Contact:** Email, phone, LinkedIn, and GitHub.
